# FZE-COMMONS

JAVA通用类库，提供如下模块:

- 基本类库和类型转换函数
- 常用的注解和接口定义
- HTTP请求库
- 领域驱动设计库
- 事件总线
- Etcd包装类及基于etcd的服务发现
- Grpc包装类
- 依赖注入工厂
- JDBC数据访问库(支持Agroal/HikariCP/C3P0)
- JWT包装类
- 通用数据查询组件
- 通用缓存存储组件
- 若干工具类

## 快速使用

使用`gradle`作为依赖管理

```kts
implementation("net.fze:fze-commons:0.4.7")
```

使用`maven`作为依赖管理

```xml
<dependency>
    <groupId>net.fze</groupId>
    <artifactId>fze-commons</artifactId>
    <version>0.4.7</version>
</dependency>
```
